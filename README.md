# Stable Matching

Gale-Shapley deferred acceptance, lattice enumeration via rotations, and hospital-resident matching.

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/stable-matching.git
cd stable-matching
make install
make run
make test
```

See [docs/](../docs/stable-matching.tex) for the math.
